Here are some projects that I have worked on in the past and am currently working on:
MusicTranscription [C, Python]:

This project was started at the hackathon, CalHacks 2014. The original goal of the project was to convert sound recordings (whether it be wav or mp3) and convert it to sheet music. Upon embarking on this project, I realized that this project was a very difficult, as polyphonic transcription remains a research topic to this day. One of the main challenges is finding the pitches played within the audio file. However, if relatively successful with recognition of pitches, there is a wide variety of applications for this project.
Price Tracker [Python, MongoDB, ExpressJS, AngularJS, NodeJS]:

This web application was developed during my internship at Symantec in the summer of 2014. Along with several graduate students, I developed a flexible price-tracking web application, generalized across multiple e-commerce platforms. Through the use of API calls and efficient web-crawling, our application kept track of prices on various Norton products on many different e-commerce platforms. One of the main take-aways from this project is the importance of optimization. This application is currently in use by Symantec's marketing team.
Image Processing and Optimization [C]:

This was a project done at one of UC Berkeley's Computer Science courses. The goal of the project was to optimize a program that could perform image convolution. By using multi-threading, and optimizing cache performance, I was able to create a fairly efficient image edge-detector, edge-sharpening, embossing, and blurring tool. Due to course policy, however, I am unable to link to the code I wrote.
